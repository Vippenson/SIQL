Supported Kinds

In the game, item kinds are coded with a number stored in 'filterFlag' field of items.

kind 'filterFlag' regular expression patterns
POWER_ARMOR"(0)"
WEAPON"(2|3|4|5)"
ARMOR"(8)"
APPAREL"(16)"
FOOD_WATER"(32|33|9)"
AID"(64|65)"
NOTES"(1024|128)"
HOLO"(65536)"
AMMO"(32768)"
MISC"(4096|266240|512|33280)"
MODS"(16384|2048)"
JUNK"(8192|270336|33792)"

Note: currently, kinds are implemented in SIQL using regular expressions (the "of" mode). If users face performance issues, I can build support for kinds into the mod.
