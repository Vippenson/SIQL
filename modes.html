Supported Comparison Modes

Syntax Meaning
FIELD"=="VALUE FIELD is equal to VALUE
FIELD"startsWith"VALUE FIELD starts with VALUE
FIELD"contains"VALUE FIELD contains VALUE
FIELD"of"VALUE FIELD is of the regular expression pattern VALUE
FIELD"<"VALUE FIELD is less than VALUE
FIELD">"VALUE FIELD is greater than VALUE
FIELD"<="VALUE FIELD is less than or equal to VALUE
FIELD">="VALUE FIELD is greater than or equal to VALUE
FIELD"!=="VALUE FIELD is NOT equal to VALUE
FIELD"!startsWith"VALUE FIELD does NOT start with VALUE
FIELD"!contains"VALUE FIELD does NOT contains VALUE
FIELD"!of"VALUE FIELD is NOT of the regular expression pattern VALUE
FIELD"!<"VALUE FIELD is NOT less than VALUE
FIELD"!>"VALUE FIELD is NOT greater than VALUE
FIELD"!<="VALUE FIELD is NOT less than or equal to VALUE
FIELD"!>="VALUE FIELD is NOT greater than or equal to VALUE
